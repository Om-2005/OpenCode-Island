## üöÄ Features

- **Notch UI**: Animated overlay that expands from the MacBook notch.
- **Summon with Hotkey**: Double-tap Command (or customize) to open the prompt interface.
- **Agent Selection**: Choose from available OpenCode agents with a simple `/` prefix.
- **Model Selection**: Pick from over 40 AI models from various providers.
- **Image Support**: Paste images using Cmd+V to include in your prompts.
- **Expandable Response Area**: Interact using an expandable area for better visibility.

## üõ†Ô∏è Usage Instructions

Once you have installed OpenCode Island, follow these steps to start using it:

1. **Launch the App**: Click on the OpenCode Island icon in your menu bar.
2. **Open the Interface**: Double-tap the Command key to summon the prompt interface.
3. **Select an Agent**: Type `/` followed by the agent's name to choose from available OpenCode agents.
4. **Choose a Model**: Select from over 40 AI models to start your interaction.
5. **Paste Images**: Use Cmd+V to insert images into the prompt area. 
6. **Interact**: Type your queries and click send. You will see responses directly in your menu bar.
